# Personal Portfolio Website

A simple, clean static website structure for a personal portfolio.

## Structure

```
website_project/
├── index.html          # Home page with hero section and featured projects
├── about.html          # About page with biography and skills
├── portfolio.html      # Portfolio grid with project showcase
├── contact.html        # Contact form and information
├── css/
│   └── style.css       # Main stylesheet with responsive design
├── js/
│   └── portfolio.js    # Portfolio filter functionality
└── images/            # Image placeholder directory
```

## Pages

### 1. **index.html** - Home Page
- Hero section with profile picture
- Featured projects grid
- Call-to-action section
- Navigation menu

### 2. **about.html** - About Page
- Personal introduction with placeholder text
- Skills list
- Experience timeline with job positions
- Professional photo

### 3. **portfolio.html** - Portfolio Page
- Project grid with category filtering
- Filter buttons (All, Web Design, Applications, Branding)
- Project cards with descriptions
- Links to project details

### 4. **contact.html** - Contact Page
- Contact form with fields (name, email, subject, message)
- Contact information section
- Social media links
- Office location details
- Map placeholder

## Images Needed

Place placeholder images in the `images/` directory:
- `profile.jpg` - Profile picture (150x150px recommended)
- `about-me.jpg` - About page image
- `project1.jpg` - Featured project 1
- `project2.jpg` - Featured project 2
- `project3.jpg` - Featured project 3
- `project4.jpg` - Portfolio project 1
- `project5.jpg` - Portfolio project 2
- `project6.jpg` - Portfolio project 3

You can use placeholder images from:
- [Unsplash](https://unsplash.com)
- [Pexels](https://pexels.com)
- [Placeholder.com](https://placeholder.com)

Example placeholder URL: `https://placeholder.com/400x300?text=Project+Image`

## Customization

1. Replace "John Doe" with your name in all HTML files
2. Update email and phone numbers in contact.html
3. Replace placeholder text with your own content
4. Update social media links in footer
5. Add your actual images to the images directory
6. Modify colors in style.css to match your branding

## Quick Start

1. Create the `images/` directory and add your images
2. Open `index.html` in a web browser
3. Test all navigation links
4. Customize content as needed
