This directory should contain all image files for the website.

Recommended image structure:
- profile.jpg (150x150px) - Your profile picture for the home page
- about-me.jpg (600x400px) - Your photo for the about page
- project1.jpg (400x300px) - Project showcase image 1
- project2.jpg (400x300px) - Project showcase image 2
- project3.jpg (400x300px) - Project showcase image 3
- project4.jpg (400x300px) - Portfolio project 1
- project5.jpg (400x300px) - Portfolio project 2
- project6.jpg (400x300px) - Portfolio project 3

You can download free placeholder images from:
- Unsplash: https://unsplash.com
- Pexels: https://pexels.com
- Pixabay: https://pixabay.com
- Placeholder Service: https://placeholder.com/400x300

Example usage in HTML:
<img src="images/profile.jpg" alt="Profile Picture">
